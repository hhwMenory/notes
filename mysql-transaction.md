### 事务隔离级别

| 隔离级别 | 脏读 | 不可重复读 | 幻读 |
| :--- | :--- | :--- | :--- |
| 未提交读 | 可能 | 可能 | 可能 |
| 已提交读 | 不可能 | 可能 | 可能 |
| 可重复读 | 不可能 | 不可能 | 可能 |
| 可串行化 | 不可能 | 不可能 | 不可能 |

``` bash

    # 脏读
        
        一个事务读取到了另外一个事务没有提交的数据
    
    # 不可重复读
    
        在同一事务中，两次读取同一数据，得到内容不同
    
    # 幻读
    
        同一事务中，用同样的操作读取两次，得到的记录数不相同 
    
```
